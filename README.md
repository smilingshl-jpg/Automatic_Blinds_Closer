# Automatic_Blinds_Closer
This repository holds all information versions and changes made in the process of making an Arduino based automatic blinds closer.
1st Version:
I want it to automatically open or close when the sun is out, using a phot resistor
2nd Version:
I want it to now be controlled by an IR Remote 
3rd Version:
I want it to connect to an app using a wifi module and then be controlled by the app and have more features
Final Features List:
  Automatic Blind Closing via app 
  Set Time for opening and closing 
  Predictive Auto Closing
